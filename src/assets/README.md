## Image Guidelines

- All images should be optimized for web (compressed)
- Use JPG for photos, PNG for logos and icons with transparency
- Maintain aspect ratios as specified
- Use high-quality images that match the Figma design

## How to Extract from Figma

1. Open the Figma design: https://www.figma.com/design/3OogQGU84ieg5xf6THLGTM/Sun-Lit-Tech
2. Select each image layer
3. Right-click → Export → Choose format (PNG for logos/icons, JPG for photos)
4. Download and place in this assets folder with the names listed above

## Fallback Handling

The code includes fallback handling for missing images:
- Colored backgrounds for icons
- Gradient backgrounds for main images
- SVG placeholders for logos
